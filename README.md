# Environment
* Python 2.7

# Setup Environment

```shell
virtualenv -p /usr/bin/python2.7 --no-site-packages ENV27
source ENV27/bin/activate
```

# How to use
1. Run the server
```
source ENV27/bin/activate
python index.py
```
2. Open the browser
3. Open http://localhost:8080/

**Port default 8080**
